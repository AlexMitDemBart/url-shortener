# url-shortener
This service will provide short aliases redirecting to long URLs. 


### Built With
* [Spring Boot](https://spring.io)
* [Flyway](https://flywaydb.org/)
* [Docker](https://www.docker.com/)
* [Lombok](https://projectlombok.org/)
